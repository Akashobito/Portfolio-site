# My Portfolio

A fully responsive personal portfolio website built with **React.js** and **Tailwind CSS**.
Designed with a mobile-first approach, it works seamlessly across all screen sizes —
mobile, tablet, and desktop.

## Tech Stack
- React.js
- Tailwind CSS
- Vite

## Features
- Responsive design (mobile + tablet + desktop)
- Clean and modern UI
- Reusable React components
- Smooth navigation

## Getting Started
npm install
npm run dev
